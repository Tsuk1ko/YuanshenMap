# 【空荧酒馆】提瓦特电子地图
食用方法：

1.点击右上角【小图标】可筛选显示物品

2.每种物品都有【序号id】，方便查缺补漏，可单击标记查看备注

3.手机放大查看更方便，简洁好用高清
